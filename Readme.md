# Betting Platform

A backend project for a modern betting application built using Node.js, SQL, and Sequelize.

This project is in its initial phase and focuses on creating a scalable, production-ready backend foundation, starting with authentication and database design.

---

## Tech Stack

- Node.js
- Express.js
- SQL (PostgreSQL / MySQL)
- Sequelize ORM
